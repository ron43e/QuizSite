Additional Directions

  • To create a quiz question, fill in the Question form, then select ONE answer format (e.g. multiple choice, true or false, or fill in the blank).
  • Finally, choose a Save option: "Save, make new question" saves a question, then instantly resets the form to add another question. "Save, go to question list" saves a question, then takes you to the Question List to see all of your questions. "Cancel, go to question list" does not save anything and takes you to the Question List.
  • Note: A question will not be saved if there is no answer format selected.
